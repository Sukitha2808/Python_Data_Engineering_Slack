Pre Order traversal:

What is Preorder Traversal?

In preorder traversal, we visit nodes in the following order:

Root

Left Subtree

Right Subtree

Often remembered as: "Root → Left → Right"

        1
       / \
      2   3
     / \  /
    4  5 6


Preorder Traversal Steps:

Start at root → 1

Go to left child → 2

Visit left child → 4

Visit right child → 5

Then go to right child → 3

Visit left child → 6

Preorder Traversal = [1, 2, 4, 5, 3, 6]


Recursive Python Implementation
===============================

class Node:
    def __init__(self, key):
        self.key = key
        self.left = None
        self.right = None

def preorder(root):
    if root:
        print(root.key, end=" ")   # 1. Visit root
        preorder(root.left)        # 2. Traverse left
        preorder(root.right)       # 3. Traverse right

# Example Tree
root = Node(1)
root.left = Node(2)
root.right = Node(3)
root.left.left = Node(4)
root.left.right = Node(5)
root.right.left = Node(6)

print("Preorder Traversal:")
preorder(root)
